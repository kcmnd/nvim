# 💤 My Neovim Configuration (Lazy.nvim, Lua-based)

Welcome to my personal Neovim configuration!  
It's written in **Lua**, uses the **Lazy.nvim** plugin manager, and is organized into a modular structure for maintainability and performance. While I primarily use it for **Python**, it should work with **Go**, **Java**, **Lua**, and web development languages like **HTML/CSS/JS**.

---

## Folder Structure

```text
nvim/
├── assets/                # Background image and other assets
├── lua/
│   ├── config/
│   │   └── lazy.lua       # Lazy.nvim setup
│   ├── kc/
│   │   └── remap.lua      # Custom keymaps
│   └── plugins/
│       └── *.lua          # Plugin definitions (one plugin per file)
├── init.lua               # Main entry point
└── lazy-lock.json         # Plugin lockfile (generated by Lazy.nvim)
```

## Features

- Written fully in Lua
- Managed with Lazy.nvim
- LSP support for multiple languages (Python, Go, Java, Lua, etc.)
- Autocompletion with nvim-cmp
- Syntax highlighting & parsing via Treesitter
- Fuzzy finding and live grep via Telescope
- File explorer with Neo-tree
- Modular setup for easy editing and extension

## Keybindings

Leader key is `<Space>`.

| Key | Mode | Action |
|-----|------|--------|
| `<Space>pv` | Normal | Open netrw file explorer |
| `<Space>ff` | Normal | Fuzzy file finder (Telescope) |
| `<Space>fg` | Normal | Live grep search (Telescope) |
| `<Space>y` | Normal | Copy LSP diagnostic under cursor to clipboard |
| `<A-j>` | Normal | Move current line down |
| `<A-k>` | Normal | Move current line up |
| `<A-j>` | Visual | Move selected lines down |
| `<A-k>` | Visual | Move selected lines up |
| `K` | Normal | Show LSP hover info (method signature, docs) |

**Note:** `<A-*>` means holding Alt (or Option on Mac). You may need to configure your terminal to send the correct Alt/Option key codes to Neovim.  

## Installation

You can install this config in a few simple steps:

```bash
# Backup your existing config if needed
mv ~/.config/nvim ~/.config/nvim.backup

# Clone this config
git clone https://github.com/kcmnd/nvim ~/.config/nvim

# Launch Neovim
nvim
```

Once Neovim opens, Lazy.nvim will automatically install the plugins.
After it's done, run the following command to finalize setup:

```bash
:Lazy sync
```

## Requirements

- **Neovim v0.10.0+** (required for treesitter and LSP features)
- **tree-sitter CLI** (required for nvim-treesitter to compile parsers):
  ```bash
  # Install via npm
  npm install -g tree-sitter-cli

  # Or via cargo
  cargo install tree-sitter-cli
  ```
- **Node.js** (for LSP servers like pyright)
- **A Nerd Font** for icon support (I use [FiraMono](https://www.nerdfonts.com/font-downloads))

## LSP Setup

For autocompletion and LSP support, install the relevant language servers:

```bash
# Python
npm install -g pyright

# Go (optional)
go install golang.org/x/tools/gopls@latest

# Other servers can be installed via Mason (:Mason) inside Neovim
```

See [nvim-lspconfig's server list](https://github.com/neovim/nvim-lspconfig/blob/master/doc/configs.md) for more options.
